## **💡1. 프로젝트 개요**

### 1-1. 프로젝트 소개
- 프로젝트 명 : 
- 서비스 MVP:
  - 영수증을 OCR로 촬영해 해당 지출 내역을 원하는 태그로 관리.
  - 태그에 맞게 지출 내역이 알아서 정리
  - 원하는 지출 내역들에 한해 자동으로 보고서 생성 가능
<br />
**기능 리스트**

- ocr → 영수증 OCR로 추출
- 태깅(만남 이름) → 영수증 OCR이 추출되면 태깅으로 분류
- 보고서 만들기
    - 태그별 보고서
        - 태그의 전체 영수증 보고서
        - 선택한 영수증 별 보고서
- 링크 공유 통한 보고서 공유
- 정산하기 버튼 클릭 시 알아서 정산 내용 정리, 공유만 하면 되도록
- 태그 별로 지출 분석을 통해 계획 및 예산 설정에 도움
    - 한 달 식사/카페/활동 비율
    - 평균 1회 만남 비용
    - 월평균 지출 예측
